# Chŏn-guk OS
**Expérience interactive compagnon — *La Sélection* — Collection Hors Jeu — Braise Noire**

---

## À propos

Chŏn-guk OS simule l'interface officielle de la République 
Démocratique et Populaire de Chŏn-guk — pays fictif 
du roman *La Sélection*.

Le joueur remplit le formulaire CIT-447, prête le Serment 
du Citoyen de Service, reçoit son attribution officielle 
— poste, logement, bloc de résidence — et commence à servir.

Ce n'est pas un tutoriel. Ce n'est pas une satire.  
C'est une simulation de ce que Jun vit depuis qu'il est né.

---
